Projects

Snek GBA C++

During the Winter quarter of 2026, I worked on a project with a peer of mine, a recreation of snake for our Systems Programming class. We used C++, butano, devcontainers (with Docker), and it runs in the browser (through mGBA). This was my first time truly working and thumbing through C++. I learned a lot about memory managment, sprites, and sprite pointers, and the value of working alongside another for a common goal!

Hangman (HTML/CSS/JavaScript)

This was a project that I worked on during the Fall quarter of 2025 for my Web Programming with JavaScript class. It's a simple iteration of the classic game Hangman, built with HTML, CSS, and JavaScript. This was my first extensive project using JavaScript, learning how to manipulate the DOM, and learning to treat elements as objects. I enjoy making a great UI/UX, I hope the design reflects as much!

Flexbox Showcase

While working with flexboxes I decided to create a flexbox showcase with all the basic fundamentals. That one would need in order to begin to create a strong foundation in organizing elements with flexbox. I hope it reflects my love for good UI/UX design!
